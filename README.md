# Payslips App

Payslip viewer with PDF download.

## Prerequisites
- Node 18+
- npm
- Xcode/Android Studio

## Setup
`npm install`

## Run
`npx react-native start`
`npx react-native run-ios`
`npx react-native run-android`

## Test
`npm test`

## Stack
- React Native CLI
- React Navigation
- Zustand
- react-native-fs
